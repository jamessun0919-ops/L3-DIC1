## ✨ 功能特色

| 功能 | 說明 |
|------|------|
| ⏰ **即時時鐘** | 24 小時制，每秒更新，顯示時、分、秒 |
| 📅 **日期顯示** | 顯示完整中文日期與星期 |
| 🌤️ **智慧問候** | 根據時段自動切換「早安 / 午安 / 晚安」 |
| 📊 **年度進度** | 計算今年已過百分比 |
| 🌌 **粒子動畫** | Canvas 繪製的動態星空粒子背景 |
| 🔮 **玻璃擬態** | Glassmorphism 風格卡片設計 |
| 📱 **響應式設計** | 完美適配桌面與行動裝置 |

---

## 🛠️ 技術架構

```
L3-DIC1/
├── index.html    # 頁面結構
├── style.css     # 樣式與動畫設計系統
├── script.js     # 時鐘邏輯與粒子動畫
└── README.md     # 專案說明
```

| 技術 | 用途 |
|------|------|
| **HTML5** | 語義化頁面結構 |
| **CSS3** | 漸層、動畫、Glassmorphism、RWD |
| **JavaScript** | 即時時鐘、Canvas 粒子系統 |
| **Google Fonts** | Outfit + JetBrains Mono 字體 |

---

## 🎨 設計亮點

- 🎆 **漸層文字** — 名字使用紫藍 → 青色與粉紅 → 紫色雙色漸層
- 💫 **浮動光球** — 三顆模糊光球在背景中緩慢漂浮
- ✨ **粒子連線** — 粒子間距離靠近時自動繪製連線
- 🃏 **卡片懸浮** — Hover 時卡片上升並發出紫色輝光
- 🕐 **分隔符閃爍** — 時鐘冒號每秒閃爍動畫

---

## 🚀 快速開始

只需在瀏覽器中開啟 `index.html` 即可運行，無需任何建置工具。

```bash
# 克隆專案
git clone https://github.com/jamessun0919-ops/L3-DIC1.git

# 開啟網頁
start index.html
```
